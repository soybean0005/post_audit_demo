Hướng dẫn gửi email tự động

Mục đích

Macro được sử dụng để tự động gửi email nhắc nhở tới Auditor khi Audit Plan (AP) chưa được gửi đúng thời hạn trước ngày audit.

Flow hoạt động của macro

  1. Đọc danh sách job trong sheet Backlog
  2. Kiểm tra từng dòng có thỏa điều kiện gửi mail hay không
  3. Lấy tên Auditor Lead
  4. Tìm email Auditor từ sheet Auditors
  5. Tạo email Outlook và gửi
  6. Đánh dấu đã gửi reminder
  7. Hiển thị tổng số email đã gửi

Các sheet liên quan

  • Sheet Backlog:
  • Sheet Auditors:

Bước 1: Xác định đối tượng nhận email và số lần nhắc nhở

Ví dụ:

  • Đối tượng nhận email: Auditor Lead
  • Số lần nhắc nhở: 1 lần
  • Trạng thái hiện tại trong bảng: AP wait for auditor

Mapping dữ liệu:

  • Tên Auditor: Sheet Backlog - Cột H
  • Email Auditor: Sheet Auditors - Cột C

Bước 2: Xác định điều kiện để gửi email nhắc nhở

Email chỉ được gửi khi thỏa tất cả các điều kiện sau:

  • Audit date còn dưới hoặc bằng 10 working days
  • Status hiện tại là 'AP wait for auditor'
  • Chưa từng gửi email reminder
  • Có thông tin Audit start date
  • Có Auditor Lead

Bước 3: Xác định nội dung email

Subject format: [Action Required - Audit Plan Submission] JobID - Client Name

Nội dung email bao gồm:

  • Job ID
  • Client Name
  • Audit Standard
  • Start Audit Date
  • Current Status
  • Yêu cầu Auditor gửi Audit Plan

Bước 4: Cơ chế chống gửi trùng email

Sau khi gửi email thành công, hệ thống sẽ ghi nhận vào cột P.

Format: Remind_dd-mmm-yyyy

Ví dụ: Remind_25-May-2026

2. Cấu trúc sheet Excel

Sheet "Backlog"

Dùng để chứa danh sách audit jobs.

Dữ liệu bắt đầu từ dòng 5.

Các cột:

  • Column A = Job number
  • Column B = Client Name
  • Column C = Standard
  • Column F = Audit start date
  • Column H = Auditor Lead
  • Column K = Current Status
  • Column O = Send CDS/client docs. to auditor (cell có giá trị nghĩa đã gửi client info cho Auditor)
  • Column P = Audit plan + Survey to client (cell có giá trị nghĩa đã gửi audit plan cho khách hàng)

Sheet "Auditors"

Dùng để mapping Auditor Name và Email.

Các cột:

  • Column B = Auditor Name
  • Column C = Auditor Email

3. Business Logic

Tạo macro tên: SendMail_When_Overdue_SendAP()

Macro sẽ loop tất cả dòng trong sheet "Backlog".

Điều kiện để gửi email:

  • Working day countdown đến audit date <= 10 ngày
  • Column K phải bằng: "AP wait for auditor"
  • Column P phải đang trống
  • Column O không được trống
  • Column H phải có Auditor Lead

Nếu thỏa điều kiện:

  • Lấy tên Auditor ở column H
  • Tìm email Auditor trong sheet "Auditors"
  • Gửi email reminder bằng Outlook
  • CC cho email Owner lấy từ Data!B1
  • Sau khi gửi thành công: ghi vào column P: Remind_dd-mmm-yyyy

Ví dụ: Remind_25-May-2026

4. Hàm tính Working Days

Tạo function: Function workingDays(startDate As Date) As Long

Code logic:

Function workingDays(startDate As Date) As Long

    workingDays = Application.WorksheetFunction.NetworkDays(startDate, Date)

End Function

Giải thích:

  • Dùng hàm NetworkDays của Excel
  • Tính số working days giữa Start Audit Date và ngày hiện tại
  • Macro phải dùng function này để check điều kiện gửi reminder

Ví dụ: CountDownDay = workingDays(ws.Cells(i, "F"). Value)

5. Hàm tìm email Auditor

Tạo function: Function GetEmailByName(ws As Worksheet, auditor As String) As String

Logic:

  • Loop tất cả dòng trong sheet Auditors
  • So sánh Auditor Name ở column B
  • Nếu match thì return email ở column C
  • Nếu không tìm thấy thì return empty string

6. Nội dung email

Subject format: [Action Required - Audit Plan Submission] JobID - Client Name

Ví dụ: [Action Required - Audit Plan Submission] AU12345 - ABC Company

HTML Body phải chứa:

  • Greeting
  • Job ID
  • Client Name
  • Audit Standard
  • Start Audit Date
  • Current Status
  • Reminder content
  • Notification section

7. Outlook Requirements

Tạo Sub: SendOutlookMail(auditorEmail As String, OwnerEmail As String, ws As Worksheet, rowNum As Long)

Yêu cầu:

  • Use CreateObject("Outlook.Application")
  • CreateItem(0)
  • Use .HTMLBody
  • Use .To
  • Use .CC
  • Có option: .Display hoặc .Send

8. Additional Requirements

  • Preserve current filter mode
  • Disable AutoFilter trước khi loop
  • Restore filter sau khi chạy xong
  • Hiển thị MsgBox:
    • số lượng mail đã gửi
    • hoặc thông báo không có job cần gửi
  • Có validation nếu Auditor Lead bị trống

9. Coding Style

  • Có comment cho từng block logic
  • Tách function rõ ràng
  • Dùng biến rõ nghĩa
  • Tối ưu để dễ maintain sau này

5.2 Tạo module

Step 1 — Open VBA Editor

  • Open Excel file
  • Press: ALT + F11 để mở VBA Editor.

Step 2 — Create Standard Module

  • Trong Project Explorer: Right-click VBAProject
  • Select: Insert → Module

Tạo module mới:

Module1

Dán các code là AI tạo vào Module:

Step 3 — Add Workbook Open Event

Trong VBA Project:

  • Double click: ThisWorkbook

Dán code:

Private Sub Workbook_Open()

    Call SendMail_When_Overdue_SendAP

End Sub

Step 4 — Save Excel File

Save file dưới format: Excel Macro-Enabled Workbook (*.xlsm)

Step 5 — Enable Macro

Khi mở file: Click: Enable Content hoặc Enable Macro để macro hoạt động.

Step 6 — Outlook Requirement

  • Outlook phải được cài trên máy
  • User phải login Outlook trước khi chạy macro
  • Nếu dùng .Send: mail sẽ gửi tự động
  • Nếu dùng .Display: mail sẽ mở draft trước khi gửi

Step 7 — Testing Guide

Khuyến nghị:

  • Test bằng .Display trước
  • Sau khi verify OK thì đổi sang .Send

Ví dụ: .Display đổi thành: .Send

Lưu ý khi sử dụng

  • Outlook phải được mở trên máy tính
  • User cần đăng nhập Outlook trước khi chạy macro
  • Tên Auditor trong sheet Backlog phải khớp hoàn toàn với sheet Auditors
  • Nếu không tìm thấy email Auditor, mail sẽ không gửi được
